# The Mirror — UI Opener Lines for Digital Assessment

**Purpose:** For the /mirror tap-through assessment, each subcategory needs ONE punchy opener line that sets up the score. This doc audits all 32 subcategories from `the-mirror-categories.md` and proposes the opener for each.

**Default UI structure per question:**
```
SUBCATEGORY NAME (display face, all-caps)
[Punchy opener line] (body face, italic)

[1] [2] [3] [4] [5]

⌄ Why this matters  →  expands to show the full workbook description
```

**Audit summary:**
- **16 subcategories** — first sentence of the workbook description works as-is
- **3 subcategories** — combine first two sentences with em-dash
- **11 subcategories** — opener should be a later sentence in the existing description (extract, no new writing needed)
- **2 subcategories** — need a new punchy line written (none of the existing sentences work standalone)

Total writing work: 2 new lines + 11 extractions + 3 combinations = 16 small edits. The other 16 are zero-work.
